Тестовое задание:
Вот структура проекта, там есть класс для работы с базой, в конструкторе впиши свои настройки подключения. Базу для задачи спроектируй сам. 
В папке /module есть папка library, нужно там собрать типа модуль управления электронной библиотекой. Считай, что пользователь уже авторизован и является администратором. В админке нужно иметь возможность создавать древовидную структуру библиотеки, добавлять в элементы этой структуры электронные книги с описанием, автором, обложкой и прикрепленным файлом самой книги. Книги нужно хранить в папке /storage/library, обрати внимание, что это непубличная папка, поэтому понадобится отдельный скрипт для скачивания файла. Сам модуль вызывается в index файле папки public, при необходимости в него же можешь подключить стили/js файлы, добавить необходимую разметку
